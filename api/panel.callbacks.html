callbacks Module¶

callbacks Module¶

Defines callbacks to be executed on a thread or by scheduling it on a running bokeh server.

class panel.callbacks.PeriodicCallback(**params)[source]¶

Bases: param.parameterized.Parameterized

Periodic encapsulates a periodic callback which will run both in tornado based notebook environments and on bokeh server. By default the callback will run until the stop method is called, but count and timeout values can be set to limit the number of executions or the maximum length of time for which the callback will run.

callback = param.Callable()

The callback to execute periodically.

count = param.Integer(inclusive_bounds=(True, True))

Number of times the callback will be executed, by default this is unlimited.

period = param.Integer(default=500, inclusive_bounds=(True, True))

Period in milliseconds at which the callback is executed.

timeout = param.Integer(inclusive_bounds=(True, True))

Timeout in seconds from the start time at which the callback expires
